# Aula-5
*Triângulo*

$A=\frac{b.h}{2}$

Sendo:

A = área

b = base

h = altura


*Quadrado*


$A=L^{2}$

Sendo:

A = área

L = lado


*Losango*

$A=\frac{D.d}{2}$

Sendo:

A = área

D = diagonal maior

d = diagonal menor



*Retângulo*

$A=b.h$

Sendo:

A = área

b = base

h = Altura


*Trapézio*

$A=\frac{(B.b).h}{2}$

Sendo:

A = área

B = base maior

b = base menor

h = altura


*circulo*

$A=\pi.r^2{2}$

sendo:

A = área

$\pi$ = constante Pi (3,14)

r = raio
